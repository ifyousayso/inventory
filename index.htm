    The Inventory

    This demo suggests a possible solution to, what I consider to be flaws of, the inventory systems of early Diablo and World of Warcraft games.

    Here, seemingly identical items in the inventory represent unique ones in order to show the functionality, else they could be stacked.

    Addressed issues

    • Not being able to pick up items due to shape limitations
    • Being able to carry too large and/or heavy items

    Usage

    • Hover the icon of an item to display information about it
    • (Shift-)Click an item in the loot frame to add a copy to the inventory
    • Drag and drop inventory items to other inventory slots or to the trash
    • Ctrl-click an inventory item to delete it
    • The inventory has a max height but can be scrolled past it using the mouse or scroll buttons
